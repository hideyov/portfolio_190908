Hideyo Uesugi

web制作を独学で修行中、Hideyo Uesugiのポートフォリオです。(2019年11月4日)

Bio

1963年　滋賀県生まれ。
1981年　広島修道高校卒業。
1981年　京都大学理学部生物学科入学。1988年卒業。
1989年から2019年に至るまで、運送会社、茅ぶき屋根工事現場の助手、インターネット・メディア、空港での機内清掃などの仕事を渡り歩く。

Web

2005年より独学でHTML/CSSを学習し、自らの音楽活動のサイトを維持・運営。2015年前後より、web開発の現場で仕事することを志向してJavaScript/PHPの学習を始めるが、独学に限界を感じ2019年5月から8月にかけて、TECH ACADEMYの「PHP/Laravelコース」「フロントエンドコース」を受講した。

DTP

Photoshop, illustratorを独学で習得し、自分のバンドのCDのジャケットデザインを数度に渡り手がけた。2014年より、陶芸や漆器などの展覧会の告知ハガキ作成を請け負う。

Movie

2012年より、音楽活動のためのyoutube動画を多数制作。2016年には山城萱葺（かやぶき）株式会社の依頼で、ヨシ焼きPR動画を作成。2018年から2019年にはインターネットメディアの会社で、取材および動画編集にも携わった。

語学・web関連資格

2013年11月　画像処理エンジニア検定エキスパート(CG-Arts協会)

SNS / Github

このサイトについて

RWD対応：768pxをブレイクポイントとするレスポンシブWebデザインを採用。

W3CのWeb標準規格に準拠： CSS Validation Service によるCSSファイルの検証、Markup Validation Service と Another HTML-lint によるHTMLファイルの検証を行ない、W3CのWeb標準規格に完全準拠するサイトを目指した。

タイルレイアウト：DTPのページではプラグイン jquery.vgrid.min.js を使って、画像をタイルレイアウトで配置した。

モーダルウィンドウ：DTP作例の画像拡大には、jQueryプラグインのLightbox2を使用した。

使用言語、ライブラリー

使用言語： HTML5, CSS3, JavaScript
使用ライブラリー、スクリプト： jQuery, Bootstrap, Lightbox2, jquery.vgrid.min.js
使用ソフト： Photoshop, Illustrator, Brackets
制作期間： 2019年9月6日〜2019年11月3日

直面した不具合

スマホの縦持ちでの画像拡大に異常：DTP作例の画像拡大には、最初、jQueryを使わない軽量スクリプトである"Luminous"を使用した。しかし、Androidのスマートフォン(Android4.4.2)、iPhone 7(iOS 12.1.4)を縦方向に持った場合のみ、DTP作例をクリックした際に、スクロールが必要なほど過大に拡大されてしまう。これがLuminousスクリプトのバグによるものなのか、現状のスキルでは確認できないため、jQueryプラグインのLightbox2に変更して解決した。

ブラウザの「戻る」ボタンでの挙動：ページ内リンクでの移動に応じてURLを変更する実装はできたが、ブラウザの「戻る」ボタンで移動すると、実際の表示とURL表示との間にずれが生じてしまう。chrome/firefox/safari/edgeいずれでも、ずれが確認された。ブラウザーの仕様であるかどうかも勉強不足のため、原因・解決方法はわからず未解決。

このサイトについて

Vue.jsの導入： ライブ情報と作品（CD）情報の多言語（日本語、英語、韓国語、中国語、スペイン語）への切り替えを、Vue.jsのv-ifディレクティブを使って実現。
また、同形式での記載を繰り返すライブ情報の部分には、Vue.jsのコンポーネント機能を利用し、サイトの維持・運営が楽に行えるようにした。

使用言語、ライブラリー、フレームワーク

使用言語： HTML5, CSS3, JavaScript
使用フレームワーク： Vue.js
使用ライブラリー： jQuery, Bootstrap, Lightbox2, slick.min.js
使用ソフト： Photoshop, Illustrator, Brackets
制作期間： 2019年8月5日〜2019年10月28日

直面した不具合

iOS 9.3.5のデバイスで閲覧できない： iPad(iOS 9.3.5)で閲覧した時、Google Chrome、safari、firefox いずれで閲覧しても、「問題が置きたため、このwebページを再度読み込みました。」等のエラーメッセージが出て、読み込めなかった。
キャッシュの削除、各ブラウザのアップデート確認（いずれも自動アップデートされて最新版）、iPadの再起動などすべて試したが、当サイトのみ、表示できない。
iPhone 7(iOS 12)では問題なく表示されたため、iOS 9.3.5 固有の不具合ではないかと思われる。あるいはVue.jsとiOS 9.3.5の相性の問題もあるのかどうか、現時点では調査方法さえもわからない。
