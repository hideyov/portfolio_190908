Hideyo Uesugi Bio

Bio： 1963年滋賀県生まれ。1981年、広島修道高校卒業。1981年、京都大学理学部生物学科入学。1988年、京都大学理学部生物学科卒業。
1989年から2019年現在に至るまで、運送会社、茅ぶき屋根工事のアシスタント、インターネット・メディア、空港での機内清掃などの仕事を渡り歩きつつ、独学でHTML/CSSを学習し、自らの音楽活動のサイトを維持・運営。
2015年ごろより、web開発の現場で仕事することを志向して、JavaScript/PHPの学習を始める。
独学に限界を感じ、2019年5月から8月にかけて、TECH ACADEMYの「PHP/Laravelコース」「フロントエンドコース」を受講。Laravel/Vue.jsという、フレームワーク学習の端緒につく。

Qualification： HTML, CSS, JavaScript
使用フレームワーク： Vue.js, Bootstrap
制作期間： 2019年8月5日〜2019年8月12日
